# XOPSWordFrequencyList

## 概要

XOPSのアドオンで使用される単語の出現頻度を算出しました。

586個のMIFファイルおよび370個のMSGファイルからテキストを抽出し、それぞれのテキストに対して形態素解析を行いました。
MIFファイルについてはブリーフィングテキストのみを処理の対象としました。

形態素解析にはMeCabを使用し、辞書にはmecab-ipadic-NEologdを採用しました。
